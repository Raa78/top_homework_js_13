## Тема
Разработка клиентских сценариев с использованием JavaScript


## Задача
### Cделать таймер
* У таймера есть (условно) две страницы:
    1. с полем ввода минут (можно дробью, чтобы были и секунды) и кнопкой старт;
    2. с кругом, который отображает визуально сколько осталось времени (100% оборота это 100% введенного времени на первом шаге) И есть две кнопки пауза и сброс.
* Пауза останавливает таймер, а сброс переводит на предыдущий этап с вводом значения.
